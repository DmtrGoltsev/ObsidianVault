# Агент Бэкенд

## Зона ответственности

- Java 21, Spring Boot 3.4.5, Spring Security, Spring Data JPA
- PostgreSQL 16, Flyway миграции
- [[REST_API]] endpoints
- [[JWT]] аутентификация
- [[FCM]] интеграция (Firebase Admin SDK)
- Планировщик: [[Reminder_Rule]], [[Recurrence_Rule]]
- [[Priority_Decay]] политика
- [[Soft_Delete]] паттерн

## Модули бэкенда

`auth`, `accounts`, `settings`, `folders`, `goals`, `tasks`, `sharing`, `calendar`, `recurrence`, `reminders`, `prioritypolicy`, `notifications`

## Ключевые файлы

- `backend/src/main/java/com/rocketflow/RocketFlowApplication.java` — точка входа
- `backend/src/main/java/com/rocketflow/auth/AuthController.java` — аутентификация
- `backend/src/main/java/com/rocketflow/tasks/TaskController.java` — CRUD задач
- `backend/src/main/java/com/rocketflow/tasks/TaskService.java` — бизнес-логика задач
- `backend/src/main/java/com/rocketflow/sharing/SharingController.java` — совместный доступ
- `backend/src/main/java/com/rocketflow/notifications/FirebaseAdminFcmSender.java` — отправка FCM
- `backend/src/main/java/com/rocketflow/notifications/ReminderNotificationScheduler.java` — планировщик уведомлений
- `backend/src/main/resources/application.yml` — конфигурация
- `backend/src/main/resources/db/migration/` — Flyway миграции

## Права на решения

- Архитектура бэкенда
- Схема БД (в рамках доменной модели)
- Выбор persistence-стратегий ([[Optimistic_Locking]], [[PostgreSQL_Advisory_Lock]])

## Активные задачи

- Поддержка текущего кода
- GHCR образ и публикация
- Staging notification smoke

## Выполненные волны

- Wave A, B, C — весь бэкенд-функционал реализован
