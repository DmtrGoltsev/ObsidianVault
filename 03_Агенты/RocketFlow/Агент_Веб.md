# Агент Веб

## Зона ответственности

- React 18, TypeScript 5, Vite 5
- react-router-dom 6, lucide-react (иконки)
- i18n локализация (русский + английский)
- Аутентификация через [[JWT]]
- Web planning flows (CRUD задач, календарь)
- Сборка и деплой статики на [[HexCore]] через Nginx

## Структура web

- `web/src/main.tsx` — точка входа
- `web/src/app/App.tsx` — корневой компонент
- `web/src/app/router.tsx` — роутинг
- `web/src/app/guards/` — защита маршрутов
- `web/src/features/auth/` — аутентификация
- `web/src/features/planning/` — планирование задач
- `web/src/features/advanced/` — продвинутые функции
- `web/src/i18n/resources.ts` — переводы
- `web/src/ui/` — UI-компоненты
- `web/src/styles/` — стили

## Ключевые API-клиенты

- `web/src/features/auth/auth-api.ts` — API аутентификации
- `web/src/features/planning/planning-api.ts` — API планирования

## Права на решения

- UI/UX дизайн (в рамках MVP3 контракта)
- Выбор UI-библиотек
- Структура feature-модулей

## Активные задачи

- MVP3 упрощение дизайна и полировка
- Адаптация под контракт [[Источник_MVP3_Контракт]]

## Выполненные волны

- Wave A: web shell, auth, i18n
- Wave B: web planning flows
- Wave C.1: scheduling authoring
