# Источник: Архитектурные решения Focus

## Тип источника

Архитектурный блюпринт — документирует ключевые архитектурные решения проекта Focus.

## Архитектурный стиль

[[Modular_Monolith]] — единый деплой-артефакт, разделённый на модули с чёткими границами. Обоснование: [[ADR_Modular_Monolith]].

## Ключевые решения

1. **Модульный монолит** — один Spring Boot jar, одна БД, модули изолированы на уровне пакетов и API
2. **JWT-аутентификация** — short-lived access token + long-lived refresh token (см. [[ADR_JWT_Auth]])
3. **Раздельный Priority Decay** — разные коэффициенты для зелёных/красных задач (см. [[ADR_Priority_Decay]])
4. **Фоновый планировщик** — Spring `@Scheduled` + `TaskScheduler` для decay, reminders, recurrence
5. **Web SPA + Android клиент** — общий REST API, бэкенд не хранит состояние UI
6. **PostgreSQL 16** — реляционная БД с JSONB для гибких настроек
7. **Flyway** — миграции БД, версионирование схемы
