# Источник: План оркестрации

`docs/19-cross-lead-orchestration-plan.md`

## Роль в проекте

Определяет cross-lead оркестрацию между ролями агентов: порядок взаимодействия, зоны ответственности, handoff-процедуры.

План координации между Backend Lead, Web Lead, Android Lead, QA Lead и DevOps Lead. Описывает зависимости между задачами, последовательность запуска волн, критерии готовности для передачи между ролями.
