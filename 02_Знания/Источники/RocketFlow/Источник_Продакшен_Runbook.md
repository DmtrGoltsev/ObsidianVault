# Источник: Продакшен Runbook

## Путь

`docs/60-hexcore-prod-runbook.md`

## Роль в проекте

Инструкция по эксплуатации production-окружения на [[HexCore]]: развёртывание, мониторинг, откат, диагностика.

## Краткое содержание

Процедуры деплоя бэкенда на HexCore (systemd unit), обновления web-статики (Nginx), проверки состояния БД (PostgreSQL 16). Команды для перезапуска, просмотра логов, отката. Контакты для эскалации. Добавлена ссылка на [[Источник_Бэкап_Runbook|backup runbook]] для процедур резервного копирования.

## Статус актуальности

Актуален.
