# Пакет контекста: Бэкенд

Выжимка знаний для [[Агент_Бэкенд|бэкенд-агента]]. Содержит ссылки на архитектуру, модули, API, миграции, тесты и CI/CD.

## Архитектурная карта
- [[MOC_Бэкенд]] — полная карта бэкенд-компонентов

## Модули
Бэкенд — [[Modular_Monolith|модульный монолит]]:
- auth — [[JWT]] аутентификация
- accounts — учётные записи
- settings — настройки
- folders — [[Folder|папки]], [[Soft_Delete]]
- goals — [[Goal|цели]], [[Soft_Delete]]
- tasks — [[Task|задачи]], [[Green_Task]], [[Red_Task]], [[Soft_Delete]]
- sharing — [[Share_Invitation]], [[Collaborator]]
- calendar — [[Planned_Time]], [[Due_Time]]
- recurrence — [[Recurrence_Rule]]
- reminders — [[Reminder_Rule]], [[FCM]]
- prioritypolicy — [[Priority_Decay]]
- notifications — [[Notification_Delivery]], [[Device_Registration]]

## API
[[REST_API]] — см. [[Источник_API_Контракты]].

## База данных
- PostgreSQL 16
- [[Flyway]] миграции (V1–V18)
- [[Soft_Delete|Мягкое удаление]] для folders/goals/tasks
- [[Optimistic_Locking|Оптимистичная блокировка]] через @Version
- [[PostgreSQL_Advisory_Lock]] для планировщика напоминаний

## Нотификации
- [[FCM]] через Firebase Admin SDK
- `FirebaseAdminFcmSender.java` + `LoggingFcmSender` (fallback)
- [[Device_Registration|Регистрация устройств]] через installationId upsert

## Тестирование
- JUnit 5 + Embedded PostgreSQL (zonky)
- CI/local verification: `mvn test` → [[Док_Backend_Тесты]], [[Док_Backend_Verification]]
- Audit inventory: 16 backend test files / 63 tests
