# Пакет контекста DevEffort

## Назначение

Дать агентам и разработчикам краткий стартовый контекст для проектирования и реализации portable JS-приложения DevEffort.

## Состав

- [[DevEffort]] — карточка проекта, архитектурные решения, гейты.
- [[Бизнес_задача_DevEffort]] — полная модель расчёта: ЕФС и Бэк, формулы, входы, выходы.
- [[Вопросы_DevEffort]] — 11 закрытых вопросов, все решены.
- `C:/Users/style/Documents/New project/outputs/КалькуляторЧП_updated.xlsx` — референсный Excel-калькулятор.

## Ключевые решения

- Критичность БЭК задаётся на весь релиз (MC+/MC/BC/ниже).
- Тип релиза: автоопределение. Hotfix если Σ(Аналитика+Разработка+Отладка+ИФТ) ≤ 5 ч-д по всем доработкам.
- ПСИ+НТ и Внедрение зависят от типа релиза (hotfix: 1+1, обычный: 3+2).
- Отладка: +20% от разработки в ЕФС, если есть смежные модули.
- Агрегация: сумма ч-д по всем доработкам на уровне релиза.
- Риск 1.35 и делитель 217 — константы MVP.
- Миграция данных пропорциональна скриптам.
- Два варианта поставки: HTML в браузере + Android PWA.

## Когда использовать

- Перед проектированием модели данных приложения.
- Перед реализацией расчётов ЕФС.
- Перед реализацией расчётов Бэк.
- Перед сравнением результатов JS-приложения с Excel-калькулятором.
