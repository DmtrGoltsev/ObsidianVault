# Recurrence Rule (Правило повторения)

## Определение

Recurrence Rule — это правило, определяющее шаблон повторения задачи. Реализовано в модуле `recurrence` бэкенда. При завершении экземпляра повторяющейся задачи система автоматически создаёт следующий экземпляр согласно правилу.

## Типы повторения

- **Ежедневно (DAILY):** задача повторяется каждый день в заданное время
- **Еженедельно (WEEKLY):** задача повторяется в выбранные дни недели
- **Ежемесячно (MONTHLY):** задача повторяется в заданное число месяца
- **Ежегодно (YEARLY):** задача повторяется в заданную дату каждый год
- **Пользовательский (CUSTOM):** cron-подобное выражение

## Свойства

- `type` — тип повторения (DAILY, WEEKLY, MONTHLY, YEARLY, CUSTOM)
- `interval` — интервал (каждые N дней/недель/месяцев)
- `days_of_week` — дни недели для WEEKLY (битовая маска)
- `day_of_month` — число месяца для MONTHLY
- `cron_expression` — cron-выражение для CUSTOM
- `end_date` — дата окончания повторений (опционально)
- `count` — максимальное количество повторений (опционально)
