# Household (семейное пространство)

**Определение:** Групповое пространство для совместного учёта финансов семьи/домохозяйства. Объединяет нескольких пользователей через механизм Membership. Все участники с активным членством (Active member) имеют доступ к shared-данным household.

**Связанные термины:**
- [[Personal_Shared]] — разделение personal/shared данных
- [[Membership]] — членство в household (invited/active/left/revoked)
