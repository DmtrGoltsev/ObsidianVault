# Device Registration (Регистрация устройства)

## Определение

Регистрация Android-устройства для приёма push-уведомлений через [[FCM|Firebase Cloud Messaging]].

## Бизнес-правила

- Привязывает устройство к аккаунту пользователя
- Хранит FCM-токен для доставки push
- Токен обновляется при переустановке приложения или смене устройства
- Используется бэкендом для адресации уведомлений конкретному устройству
- Модуль бэкенда: `notifications` (источник: `docs/04-architecture-blueprint.md`)

## Связанные термины

- [[FCM]] — канал доставки push
- [[Notification_Delivery]] — запись о доставке
- [[Reminder_Rule]] — правило, порождающее уведомление
