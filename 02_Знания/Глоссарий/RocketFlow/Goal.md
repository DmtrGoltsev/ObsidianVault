# Goal (Цель)

## Определение

Осмысленный среднесрочный или долгосрочный результат, принадлежащий папке. Группирует связанные задачи вокруг одного результата.

## Бизнес-правила

- Принадлежит ровно одной [[Folder|папке]]
- Принадлежит ровно одному владельцу
- Может содержать несколько [[Task|задач]]
- Может быть расшарена другому пользователю через [[Share_Invitation]]

## Пример использования

- «Повышение» — цель в папке Work
- «Расширение команды» — цель в папке Work
- «Полумарафон» — цель в папке Health

## Связанные термины

- [[Folder]] — родительский контейнер
- [[Task]] — единица работы внутри цели
- [[Share_Invitation]] — механизм доступа к цели
- [[Collaborator]] — соавтор с доступом к цели
