# Tag (Тег)

## Определение

Переиспользуемая метка задачи для гибкой группировки вне целей и папок.

## Бизнес-правила

- Теги ограничены пользователем в MVP (user-scoped)
- Один тег может быть присвоен многим [[Task|задачам]]
- Задача может иметь множество тегов

## Пример использования

- `finance` — группировка финансовых задач
- `hiring` — задачи найма
- `urgent` — срочные задачи

## Поля

- id, owner_user_id, name, color (опционально)

## Связанные термины

- [[Task]] — задача, к которой привязан тег
- [[Folder]] — контейнер (теги работают поверх папок)
- [[Goal]] — цель (теги работают поверх целей)
