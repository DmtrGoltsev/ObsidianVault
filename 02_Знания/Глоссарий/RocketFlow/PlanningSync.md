# PlanningSync

## Определение

PlanningSync — механизм офлайн-синхронизации планирования между Android-клиентом и бэкендом. Реализован через `PlanningSyncWorker` (WorkManager) и `ConflictResolver`. Позволяет пользователю планировать задачи без подключения к сети с последующей синхронизацией при появлении соединения.

## Бизнес-правила

- Планирование работает офлайн (локальное хранилище Android)
- При появлении сети запускается `PlanningSyncWorker`
- `ConflictResolver` разрешает конфликты версий (последнее изменение побеждает)
- Основан на [[Optimistic_Locking|оптимистичной блокировке]] версий
