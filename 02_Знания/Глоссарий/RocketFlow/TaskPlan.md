# TaskPlan

## Определение

TaskPlan — план задач с приоритетной сортировкой. Используется в Android-модуле `planning`. Представляет упорядоченный список задач, отсортированный по приоритету с учётом [[Priority_Decay|снижения приоритета]] для просроченных задач.

## Бизнес-правила

- Сортировка по убыванию приоритета
- Красные задачи выше зелёных при равном приоритете
- Поддерживает drag-drop переупорядочивание
- Синхронизируется с бэкендом через [[PlanningSync]]

## Связанные термины

- [[Green_Task]] — зелёная задача
- [[Red_Task]] — красная задача
- [[Priority_Decay]] — снижение приоритета
- [[PlanningSync]] — синхронизация плана
