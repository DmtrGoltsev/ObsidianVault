# FolderNote

## Определение

FolderNote — заметка в папке. Относится к модулю `notes` бэкенда. Замена устаревшего модуля `folder_notes`. Представляет текстовую заметку, прикреплённую к [[Folder|папке]], с базовым форматированием.

## Бизнес-правила

- Принадлежит ровно одной [[Folder|папке]]
- Заменяет функциональность `folder_notes` из ранних версий
- Может быть связана с другими сущностями через [[EntityLink]]
- Поддерживает [[Soft_Delete|мягкое удаление]]

## Связанные термины

- [[Folder]] — родительская папка
- [[EntityLink]] — связь с другими сущностями
- [[Idea]] — идея (другой тип заметки в папке)
