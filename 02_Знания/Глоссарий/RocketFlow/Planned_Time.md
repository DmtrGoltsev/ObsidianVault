# Planned Time (Плановое время)

## Определение

Дата и время, когда пользователь намеревается работать над [[Task|задачей]]. Опциональное поле задачи.

## Бизнес-правила

- Устанавливается пользователем
- Хранится в часовом поясе пользователя (канонический — из настроек аккаунта)
- Используется для планирования и напоминаний
- Может быть изменено через [[Reschedule_Event|перенос]]

## Связанные термины

- [[Due_Time]] — крайний срок (отдельное поле)
- [[Reschedule_Event]] — запись переноса планового времени
- [[Reminder_Rule]] — правило напоминания на основе планового времени
- [[Task]] — задача, к которой привязано плановое время
