# Soft Delete (Мягкое удаление)

## Определение

Паттерн, при котором запись не удаляется физически из БД, а помечается как архивная (флаг `archived`). Применяется к сущностям RocketFlow: [[Folder]], [[Goal]], [[Task]].

## Бизнес-правила

- У всех ключевых сущностей есть поле `archived_flag` (опционально)
- Архивная запись не отображается в активных списках
- Возможно восстановление из архива (предположение)
- Физическое удаление не применяется в штатном режиме

## Пример использования

- Пользователь «удаляет» папку — она помечается archived, цели и задачи внутри не теряются
- Задача завершена и archived — не показывается, но хранится для истории
