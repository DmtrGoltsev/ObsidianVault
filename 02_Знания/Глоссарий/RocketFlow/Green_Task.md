# Green Task (Зелёная задача)

## Определение

Задача, выполнение которой создаёт положительный прогресс. Тип `green` в доменной модели [[Task]].

## Бизнес-правила

- Если выполнена — пользователь движется к успеху
- Если не выполнена — состояние остаётся в основном прежним
- Имеет собственные настройки [[Priority_Decay|снижения приоритета]], отдельные от красных задач (источник: `docs/03-domain-specification.md`)

## Пример использования

- «Подготовить план повышения» — зелёная задача: выполнение ведёт к росту
- «Пройти курс» — зелёная задача: выполнение даёт новые навыки

## Связанные термины

- [[Task]] — родительская сущность
- [[Red_Task]] — противоположный тип
- [[Priority_Decay]] — политика снижения приоритета
