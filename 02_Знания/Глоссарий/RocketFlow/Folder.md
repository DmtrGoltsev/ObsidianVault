# Folder (Папка)

## Определение

Контейнер верхнего уровня для группировки целей. Принадлежит одному пользователю и не может быть расшарена в MVP.

## Бизнес-правила

- Принадлежит ровно одному владельцу
- Может содержать несколько целей
- Не подлежит совместному доступу в MVP
- Не является целью сама по себе

## Пример использования

```
Work → Содержит цели «Повышение», «Расширение команды»
Family → Содержит цель «Полумарафон»
Health → Содержит цель «Запись к стоматологу»
```

## Связанные термины

- [[Goal]] — цель внутри папки
- [[Task]] — задача внутри цели
- [[Collaborator]] — соавтор (не применим к Folder в MVP)
