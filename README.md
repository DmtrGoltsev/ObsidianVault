# Obsidian Vault — база знаний проектов

> **Открой первым:** [[Старт_Агента]] — точка входа для агентов.

## Что здесь

Это Obsidian vault для многоагентной разработки. Хранит цель проектов, контекст, задачи, решения, источники, доказательства, промпты, регламенты и связи между ними.

## Быстрый старт

1. Открой [[Старт_Агента]] — все правила, карта vault, навигация
2. Открой [[MOC_Все_Проекты]] — хаб всех проектов
3. Выбери проект → открой его MOC-карту

## Активные проекты

- [[Finance]] — ручной учёт личных и семейных финансов (скелет)

## Принципы

- Русский язык основной, English для тех. терминов
- YAML frontmatter обязателен
- Код не копировать — ссылки на файлы репозитория
- Секреты не сохранять
- Факты с источниками и доказательствами
- Vault = Git-репозиторий: работай в отдельных ветках
