# Промпт: Деплой Finance

## Контекст

Production сервер: **HexCore** (45.10.110.42). nginx, systemd, PostgreSQL 16. Finance изолирован от RocketFlow: свои routes, service, database.

## Расположение компонентов

| Компонент | Путь | Примечание |
|---|---|---|
| Backend | `/opt/finance/current` | uvicorn `:8081`, systemd service `finance-backend` |
| Frontend | `/var/www/finance/current` | статика, nginx location `/finance/` |
| API proxy | nginx location `/finance-api/` | `proxy_pass http://127.0.0.1:8081/` |
| Database | PostgreSQL `finance` | migrations через Alembic |
| OCR | Tesseract `/usr/bin/tesseract` | `rus+eng`, включён через env |

## Процедура деплоя

1. **Build frontend**:
   ```bash
   cd apps/web-pwa
   VITE_BASE_PATH=/finance/ VITE_API_BASE_URL=/finance-api npm run build
   ```
2. **Stop backend service**: `sudo systemctl stop finance-backend`
3. **Copy backend** → `/opt/finance/current` (replace artifact, preserve `.env`)
4. **Copy frontend** → `/var/www/finance/current`
5. **Run Alembic migrations**:
   ```bash
   cd /opt/finance/current
   alembic -c alembic.ini upgrade head
   ```
6. **Start backend**: `sudo systemctl start finance-backend`
7. **Verify** (см. ниже)

## Проверка после деплоя

```bash
curl -fsS http://127.0.0.1:8081/health
curl -I http://127.0.0.1/finance/
curl -I http://127.0.0.1/finance/manifest.webmanifest
curl -I http://127.0.0.1/finance/sw.js
curl -I http://127.0.0.1/finance-api/health
```

- `index.html` ссылается на assets под `/finance/`.
- `manifest.webmanifest`: `start_url` и `scope` = `/finance/`.
- Service worker: `/finance/sw.js`, scope `/finance/` (не `/`).
- API calls идут на `/finance-api/api/v1/...` (не `/api/`).
- OCR: authenticated запрос PNG на `/finance-api/api/v1/capture-drafts/screenshot-ocr` возвращает draft, не error.

## Правила

- НЕ деплоить без успешного `health` check.
- НЕ трогать RocketFlow routes, services, databases, nginx locations.
- НЕ запускать `dev_seed` в production.
- Все env-переменные backend с префиксом `FINANCE_BACKEND_*`.
- Перед деплоем убедиться: миграции, секреты, backend health, frontend build, QA login smoke — пройдены.
