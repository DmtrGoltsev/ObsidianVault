## Как добавить новый проект

1. Создать подпапки проекта во всех категориях:
   - `01_Работа/Проекты/ИмяПроекта/`
   - `01_Работа/Задачи/ИмяПроекта/`
   - `02_Знания/Глоссарий/ИмяПроекта/`
   - `02_Знания/Источники/ИмяПроекта/`
   - `02_Знания/Доказательства/ИмяПроекта/`
   - `02_Знания/Пакеты_контекста/ИмяПроекта/`
   - `03_Агенты/ИмяПроекта/`
   - `04_Решения/ИмяПроекта/`
   - `05_Система/Карты_MOC/ИмяПроекта/`
   - `05_Система/Регламенты/ИмяПроекта/` (если нужны)
   - `05_Система/Схемы/ИмяПроекта/` (если нужны)
   - `05_Система/Промпты/ИмяПроекта/` (если нужны)
2. Создать главную заметку проекта в `01_Работа/Проекты/ИмяПроекта/`
3. Добавить ссылку на проект в секцию «Активные проекты» этой карты
4. Создать MOC-карту проекта в `05_Система/Карты_MOC/ИмяПроекта/`
5. При необходимости добавить термины в `02_Знания/Глоссарий/ИмяПроекта/`
