# Finance — Развертывание на HexCore

## Сервер

```
HexCore (45.10.110.42)
│
├── Nginx (:80 / :443)
│   │
│   ├── /finance/ ──────► /var/www/finance/current/ (React PWA статика)
│   │                      ├── index.html (base /finance/)
│   │                      ├── manifest.webmanifest (scope /finance/)
│   │                      └── sw.js (scope /finance/)
│   │
│   ├── /finance-api/ ──► http://127.0.0.1:8081 (reverse proxy)
│   │                      └── client_max_body_size 8m (для OCR upload)
│   │
│   ├── / ──────────────► RocketFlow frontend (соседний проект)
│   └── /api/ ──────────► http://127.0.0.1:8080 (RocketFlow backend)
│
├── systemd: finance-backend.service
│   └── uvicorn (Python / FastAPI, :8081)
│
├── PostgreSQL 16 (:5432)
│   ├── database: finance
│   ├── user: finance_app
│   └── 9 миграций (Alembic, external policy)
│
├── /opt/finance/current/          — backend code
├── /etc/finance/backend.env       — конфигурация (FINANCE_BACKEND_*)
└── /usr/bin/tesseract             — OCR для PWA / iOS
```

## Переменные окружения (ключевые)

```
FINANCE_BACKEND_ENVIRONMENT=production
FINANCE_BACKEND_DATABASE_URL=postgresql+asyncpg://finance_app:<secret>@127.0.0.1:5432/finance
FINANCE_BACKEND_AUTH_COOKIE_PATH=/
FINANCE_BACKEND_AUTH_COOKIE_SECURE=true
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_ENABLED=true
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_TESSERACT_CMD=/usr/bin/tesseract
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_LANG=rus+eng
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_MAX_UPLOAD_BYTES=8388608
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_MAX_PIXELS=16000000
FINANCE_BACKEND_CAPTURE_SCREENSHOT_OCR_TIMEOUT_SECONDS=8
```

## Frontend-сборка

```
cd apps/web-pwa
VITE_BASE_PATH=/finance/ VITE_API_BASE_URL=/finance-api npm run build
└── output → /var/www/finance/current/
```

## Isolation-контракт с RocketFlow

```
Finance НЕ трогает:
  ├── /         — RocketFlow frontend
  ├── /api/     — RocketFlow backend (127.0.0.1:8080)
  ├── RocketFlow service / DB / roles / dirs

RocketFlow НЕ трогает:
  ├── /finance/     — Finance frontend
  └── /finance-api/ — Finance backend (127.0.0.1:8081)
```

## Verification-чеклист

```
1. curl -fsS http://127.0.0.1:8081/health               (backend alive)
2. curl -I http://127.0.0.1/finance/                     (frontend served)
3. curl -I http://127.0.0.1/finance/manifest.webmanifest (manifest ok)
4. curl -I http://127.0.0.1/finance/sw.js                (SW scope ok)
5. curl -I http://127.0.0.1/finance-api/health           (proxy ok)
6. curl http://127.0.0.1/          && curl http://127.0.0.1/api/  (RocketFlow intact)
```
