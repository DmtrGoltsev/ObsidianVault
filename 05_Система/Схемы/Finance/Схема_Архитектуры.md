# Finance — Архитектура системы

## Клиентский слой

```
Пользователь (Browser / Android / iPhone)
    │
    ├── Browser ──► Nginx (/finance/) ──► React PWA статика
    │                            │
    │                            └── /finance-api/ ──► FastAPI (:8081)
    │
    ├── Android ──► FastAPI (:8081) [Bearer auth]
    │            │
    │            └── ML Kit OCR (on-device, без загрузки на сервер)
    │
    └── iPhone ──► Capacitor ──► Nginx ──► FastAPI (:8081)
```

## Backend (FastAPI :8081)

```
FastAPI (:8081)
│
├── Auth
│   ├── Browser/PWA: cookie + CSRF + refresh token
│   └── Android/iOS: Bearer token + refresh
│
├── Доменные модули
│   ├── Accounts        — CRUD счетов (personal / shared)
│   ├── Categories      — CRUD категорий + иконки
│   ├── Transactions    — доход / расход / перевод / брокер
│   └── Reports         — AnalyticsView (shared family / combined viewer)
│
├── Capture Drafts (OCR → Draft → Confirm → Transaction)
│   ├── /screenshot-ocr — загрузка скриншота, server-side OCR
│   ├── /capture-drafts — CRUD черновиков, confirm/discard
│   └── /category-mappings — маппинг внешних меток на категории
│
└── PostgreSQL 16 (:5432)
    ├── database: finance
    ├── 9 миграций (Alembic)
    └── sourceType: manual (все операции — пользовательские)
```

## OCR-маршрутизация

```
OCR стратегии по платформе:
│
├── Android ─── ML Kit (on-device)
│                └── кандидат отправлен как POST /api/v1/capture-drafts
│
└── PWA / iOS ── Tesseract (server-side)
                  └── POST /api/v1/capture-drafts/screenshot-ocr (upload)
                      └── PNG / JPEG / WebP, max 8 MB, max 16 Mpx
```

## Доменная модель (связи)

```
User 1 ──► 0..many Membership
Household 1 ──► 1..2 active Membership  (MVP: max 2 участника)
User 1 ──► 0..many personal Account
Household 1 ──► 0..many shared Account
Account 1 ──► 0..many Operation
Operation 0..many ──► 0..1 counterparty Account (переводы)
Category 1 ──► 0..many Operation
Icon 1 ──► 0..many Category
```

## Правила видимости

| Объект            | Кто видит                         |
| ----------------- | --------------------------------- |
| Personal Account  | только владелец                   |
| Shared Account    | активные участники Household      |
| Operation         | наследуется от Account            |
| Personal Category | только владелец                   |
| Household Category| активные участники Household      |
| shared family report    | только shared-счета household |
| combined viewer overview| shared + свои personal       |
