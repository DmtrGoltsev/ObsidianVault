# Схема ветвления RocketFlow

## Текущие ветки

| Ветка | Назначение | Статус |
|-------|-----------|--------|
| `MVP3` | Активная разработка: упрощение дизайна, полировка UX, фиксы Android | **Активна** — tracking `origin/MVP3`, HEAD `21f95c1`, 23 коммита относительно `MVP2` |
| `MVP2` | Завершённая стадия MVP2: иерархия, связи, заметки | Завершена |
| `release_1` | Первый production-релиз, деплой на HexCore | Заморожена |
| `backup/release_1-before-rollback-20260518` | Бэкап состояния перед откатом | Архив |
| `master` | Историческая основная ветка | Устарела |

## Стратегия

```
master ──────► MVP2 ──────► release_1 (production)
                    │
                    └──────► MVP3 (активная разработка)
                                 │
                                 └──────► agent/opencode/* (ветки агентов)
```

### Правила
- **Основная разработка** — ветка `MVP3`
- **Production-релизы** — `release_1` (деплой через GitHub Actions при push)
- **Агенты** — работают в `agent/<среда>/<задача>`, пушат PR в `MVP3`
- **Merge** — выполняет оркестратор после проверки

### Автодеплой (из `docs/58-github-cicd-policy.md`)
- Push в `MVP2` → автоматический деплой на production
- Push в `release_1` → автоматический деплой на production
- `MVP3` — без автодеплоя (ручной запуск)
