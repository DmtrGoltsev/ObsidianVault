# Задача: Staging-сертификация нотификаций

## Описание

Провести сертификацию нотификаций на staging-окружении: подтвердить, что reminder → push → tap → task open работает стабильно в окружении, приближенном к production.

## Контекст

Smoke-процедура нотификаций доказана локально ([[Док_Нотификации_E2E]]). Требуется подтверждение на staging-окружении перед релизом.

## Критерии приёмки

- [ ] Staging-окружение развёрнуто с реальным [[FCM]] (не LoggingFcmSender)
- [ ] Push-уведомление доставлено на физическое/эмуляторное Android-устройство
- [ ] Тап по уведомлению открывает целевую задачу
- [ ] Задержка доставки в допустимых пределах (< 30 сек)
- [ ] Результат задокументирован
