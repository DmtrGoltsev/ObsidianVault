# DevEffort

## Цель

Создать portable JS-приложение для оценки трудозатрат IT-продуктов по релизам. Два варианта поставки: HTML в браузере (desktop) + Android PWA.

## Стек

- JavaScript (vanilla).
- HTML-файл, открываемый в браузере.
- Android PWA как второй вариант поставки.
- Данные сохраняются через скачивание JSON.

## Текущий статус

Бизнес-логика проанализирована и задокументирована. Все вопросы закрыты. Готово к проектированию модели данных и прототипированию.

## Команда

- style — владелец проекта.
- Codex — агентная поддержка анализа и ведения контекста.

## Окружения

- Локальная папка запуска приложения.
- Excel-калькулятор как референс модели расчёта.

## Архитектурные решения

| Решение | Выбор |
|---------|-------|
| Desktop-поставка | HTML-файл в браузере |
| Mobile-поставка | Android PWA |
| Данные | JSON-экспорт/импорт |
| Риск | 1.35 — константа MVP |
| Делитель (пше/год) | 217 — константа MVP |
| Критичность БЭК | На весь релиз |
| Тип релиза | Авто: hotfix если Σ(Аналитика+Разработка+Отладка+ИФТ) ≤ 5 ч-д |
| Агрегация | Сумма ч-д по всем доработкам на уровне релиза |
| Отладка | +20% от разработки, если есть смежные модули (ЕФС) |
